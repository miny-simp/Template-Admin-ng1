# Template-Admin-ng1
**运行项目需要执行的命令：**
 - npm install -g grunt-cil
 - npm install -g cnpm --registry=https://registry.npm.taobao.org
 - grunt serve
 
 **这是大漠穷秋的一个代码模板**
  - README-NiceFish.md 是大漠穷秋关于这个项目的描述
  - 这是一个后台管理模板项目，纯前端代码，没有后端，目的在于为使用Angular 1.x的开发者提供一个最佳实践。
